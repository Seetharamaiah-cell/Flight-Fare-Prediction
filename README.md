✈️ Flight Fare Prediction using Machine Learning

Predict flight ticket prices based on travel details using a machine learning model. This interactive web application is built with Streamlit and powered by XGBoost, allowing users to estimate airfare quickly and easily.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

📌 Project Overview

This project predicts flight fares by analyzing factors such as airline, source, destination, departure time, arrival time, duration, total stops, and travel date. It demonstrates the complete machine learning workflow, from data preprocessing and feature engineering to model training and deployment with Streamlit.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🚀 Features
✈️ Predict flight ticket prices in real time
📊 Interactive and user-friendly Streamlit interface
🧹 Data preprocessing and feature engineering
🤖 Machine Learning model using XGBoost
📈 Fast and accurate fare prediction
🌐 Ready for deployment on Streamlit Community Cloud
